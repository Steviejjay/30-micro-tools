# Fertilizer Dosage Calculator

Calculate fertilizer quantities needed for agricultural fields.

## Problem It Solves
Helps farmers determine accurate fertilizer amounts, preventing over-application (waste/pollution) or under-application (poor yields).

## How It Works
- **Inputs:** Field area (sqm), fertilizer type, number of applications
- **Calculation:** (Area × Rate per sqm × Applications) ÷ 1000
- **Output:** Total fertilizer needed in kg

## Example
- **Input:** 1000 sqm, NPK 15-15-15 (20g/sqm), 3 applications
- **Calculation:** (1000 × 20 × 3) ÷ 1000 = 60 kg
- **Output:** 60 kg fertilizer needed
